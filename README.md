

# 📌 Project Overview

This project focuses on detecting rock fractures from images using a Convolutional Neural Network (CNN). The model is trained on a dataset of 40,000 labeled images to classify the presence or absence of fractures.

# ⚙️ Methodology
Data Preparation: Data loading, verification, and train/test split
Preprocessing: Image transformations and batch creation using DataLoaders
Modeling: Design and implementation of a CNN architecture
Evaluation: Performance assessment and model optimization

# 🎯 Objective

Build a robust image classification model capable of accurately identifying rock fractures from visual data.
